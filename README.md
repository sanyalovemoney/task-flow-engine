# Task-Flow Engine 

Система автоматизованої обробки черги завдань, розроблена на Node.js. Проект демонструє використання сучасних паттернів проектування, асинхронності та структур даних.

##  Опис проекту
Цей проект є консольним двигуном, який генерує, сортує та обробляє завдання з урахуванням пріоритетів, логування та безпеки. Кожен модуль системи відповідає окремому технічному завданню (Task).

##  Стек
- **Runtime:** Node.js 
- **Мова:** JavaScript 
- **Інструменти:** Git, NPM
